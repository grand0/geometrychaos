Editor elements

Overview

This is the starting screen of an editor:

Big rectangle in the center is the game field. It will help you visualize where objects are placed and how they look.

At the bottom of the screen you can see the timeline. It shows when objects are appearing and hiding.

Between game field and timeline are timeline controls. They allow you to play/pause level, add new objects and seek through timeline.

Located on the right side is object editor. It allows to change properties of selected object, such as name, shape, timings, keyframes and more.

At the very top is menu bar. You can use it to load or save map, and also to add music to your map.

Read further to learn more about each of the elements of the editor.

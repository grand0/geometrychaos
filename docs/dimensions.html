Dimensions

To be able to better control the placement of objects, you might want to know about dimensions and sizes of objects and game field.

Game field

The game field is 1280x720 pixels (they are not really pixels, rather units because of all that scaling stuff, but here I will call them pixels). The center of the game field has coordinates (0, 0). Y-axis of the field is inverted from the normal XY-axes, so the top right corner of the field has coordinates (640, -360).

Objects

The default size of objects is 100x100 pixels (not really pixels, you got the point). That's, however, is not true for all the shapes, so here is the list of all shapes, their sizes and origins:

  • Square: 100x100, origin at the center.

  • Circle: 100 in diameter, origin at the center.

  • Triangle: side length = 100, which results in height length = ~86.6025, origin is at the centre of gravity. Angles coordinates: (0, -57.735), (50, 28.8675), (-50, 28.8675).

  • Hexagon: diagonal = 100, side length = 50, height = 86.6025, origin at the center. I'm too lazy to calculate all angles, I'm sure you will figure it out.

  • Text: size of the font is 32pt, length and origin are variable.

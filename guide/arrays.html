Arrays

Arrays are a powerful feature that you can use to create repetitive patterns or particles. Array is basically an object duplicated many times with possible modifications to its properties.

Say, for example, you want to create a set of particles that looks something like this:

Of course, you can create this animation by hand, duplicating one object and changing properties of each of them, but in case like this it is much more convenient to use arrays.

To create an array, you must first create an object which will be duplicated. In this example it is the circle which goes up and near the end grows and fades out. I placed it on the left of the screen, so that future duplicates can propagate to the right. Select the object and press Ctrl+Shift+D. A new window will appear:

Count and Arrays count values control how much objects will appear after arrays creation, Interval controls the distance between objects appearing in milliseconds. I don't really know how to explain all this in words, so here is the picture:

Notice that original object gets included in count of objects, so that in every array there is an equal amount of objects.

Per array deltas controls how properties are changed between duplicates. For example, if PosX 'x' delta (first column) is set to 30, and original object has keyframe with PosX value of 0, then the original object will keep its keyframe with value of 0, the second duplicate will get a keyframe with value of 30, the third duplicate will get a keyframe with value of 60, and so on. If there are several arrays, then each array will have similar set of objects. 'y' delta (second column) does the same thing but between arrays (from top to bottom).

Notice that all created objects have the same name. This means that you can use Shift+Delete hotkey to remove all objects in the array, except the selected one.

Geometry Chaos Editor Help

Getting started

Geometry Chaos features map editor. Maps created in this editor can be played and shared with others.

Map vs. Level

This guide uses terms map and level. Level is a collection of game objects, their properties and other entities, which you see on screen while playing. Map is a collection of assets, typically level and audio file.

On the lower level, a map is a .gcmap file which is a zip file in disguise, which contains all the assets that are necessary to play it, including the level file, which consists of JSON code.

Running editor

Editor comes as a standalone application. You can get latest release and instructions on how to build and run it at the GitHub page.

Notice that this guide is for the 1.2 version of the game. Make sure you are using the latest version!
